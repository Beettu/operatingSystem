# operatingSystem
This repository contains Operating System (OS) lab programs written in the C programming language. The programs demonstrate different CPU scheduling algorithms and important Operating System concepts commonly studied in academic courses.

📚 About the Repository

This repository is created to organize and maintain various Operating System laboratory programs. Each program is designed to help understand how scheduling algorithms and other OS mechanisms work internally.

These implementations are useful for students who want to learn how theoretical OS concepts are translated into actual code.

💻 Programming Language

C

🛠 Requirements

To compile and run these programs, you need:

GCC Compiler

Linux / Windows / macOS Terminal

Basic understanding of C programming

▶️ How to Compile and Run

Compile the program:

gcc filename.c

Run the program:

./a.out

Example:

gcc priority.c
./a.out
📂 Programs Included

The repository currently includes implementations of the following algorithms:

Priority Scheduling

First Come First Serve (FCFS) Scheduling

Shortest Job First (SJF) Scheduling

Round Robin Scheduling

Additional Operating System algorithms may be added in the future.

📖 Example Program

The repository includes an implementation of the Priority Scheduling algorithm in C.
The program calculates and displays:

Waiting Time

Turnaround Time

Order of Process Execution

🎓 Academic Purpose

This repository is developed for Operating System laboratory practice and to support academic learning. The programs aim to provide practical understanding of CPU scheduling techniques.
